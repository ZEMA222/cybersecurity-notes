🔵 i - Configuring Bash
يعني: إزاي تزبط بيئة الشغل بتاعتك في الـ Bash Shell بحيث تشتغل بسرعة وكفاءة وسلاسة.

✅ 1. PATH Environment Variable
📌 يعني إيه؟
$PATH هو متغير بيئي بيقول للـ Shell:

"لما تكتب أمر (زي ls أو python)، فتّش عليه فين؟"

🧪 مثال:
echo $PATH
🔹 هيطبع مسارات زي:
/usr/local/sbin:/usr/local/bin:/usr/bin:/bin:/home/mazen/scripts
📌 ليه مهم؟
لو عندك سكريبت أو برنامج جديد، وعايز تشغله من أي مكان، تحط مساره في $PATH.

🧪 مثال عملي:
export PATH=$PATH:/home/mazen/mytools
كده أي سكريبت جوه mytools/ تقدر تشغله من أي مكان.

✅ 2. history Command
📌 يعني إيه؟
بيعرضلك كل الأوامر اللي كتبتها قبل كده في التيرمنال.

🧪 مثال:
history
🔹 هيطبع:
1  cd Downloads
2  nano test.txt
3  sudo apt update
📌 ليه مفيد؟
تشوف أنت عملت إيه.

تعيد تشغيل أوامر طويلة بسهولة.

✅ 3. which Command
📌 يعني إيه؟
بيقولك الأمر اللي بتكتبه جاي منين.

🧪 مثال:
which python
🔹 النتيجة:
/usr/bin/python
📌 ليه مهم؟
علشان تتأكد أنت بتستخدم أي نسخة من البرنامج، خصوصًا لما يكون فيه أكتر من نسخة مثبتة.

✅ 4. Re-executing commands (!!)
📌 يعني إيه؟
تشغّل آخر أمر كتبته.

🧪 مثال:
!!    # يعيد تنفيذ آخر أمر
!ls   # يعيد آخر أمر بدأ بـ ls
📌 مفيد إمتى؟
لو نسيت تكتب sudo مثلاً:
apt update   # هيقولك permission denied
sudo !!      # يعيد الأمر اللي فات بـ sudo
✅ 5. Creating Aliases
📌 يعني إيه؟
الـ Alias = اسم مختصر لأمر كبير.

🧪 مثال:
alias cls='clear'
alias ll='ls -la --color=auto'
دلوقتي لو كتبت ll، هيشتغل كأنك كتبت ls -la --color=auto

✅ 6. Displaying Aliases
alias      # يعرض كل aliases اللي عندك
✅ 7. Deleting Aliases
unalias ll
✅ 8. Customizing the Prompt
📌 يعني إيه؟
PS1 هو المتغير اللي بيتحكم في شكل الـ prompt (السطر اللي بتكتب فيه الأوامر).

🧪 مثال:
export PS1="\u@\h:\w$ "
🔹 ده هيخلي الباش يظهر كده:
mazen@kali:~/Desktop$
📌 الرموز المهمة:
الرمز	المعنى
\u	اسم المستخدم
\h	اسم الجهاز
\w	المسار الحالي
