		# kali linux shortcuts
			- CTRL SHIFT + ==> Zoom in
			- CTRL - ==> Zoom Out

			- CTRL SHIFT R ==> Split Horzointally

			- CTRL SHIFT E ==> Close sub-terminal

			- CTRL SHIFT C ==> Copy
			- CTRL SHIFT V ==> Paste
			- CTRL C ==> Kill or stop
			- tab
		# kali linux commands
			pwd: بتجيب المكان اللي انت واقف فيه حاليًا (الدايركتوري الحالي).
ls : بتعرض كل الملفات والفولدرات الموجودة في المكان الحالي.
cd : بتنقلك لفولدر معين.
cd .. : بتطلعك خطوة لفوق (للفولدر الأب).
mkdir : بتعمل فولدر جديد.
touch : بتعمل ملف جديد فاضي.
nano : محرر نصوص، بـ (CTRL + S) تحفظ و(CTRL + X) تخرج.
cat : بتعرض محتويات الملف مباشرة في التيرمنال.
rm : بتحذف ملف معين.
rmdir :    بتحذف فولدر فاضي.
rm -R ...     : بتحذف فولدر بكل اللي جواه.
cp           : بتنسخ ملف أو فولدر لمكان تاني.
mv      : بتنقل ملف أو فولدر أو بتغير اسمه.
chmod   : بتغير صلاحيات الملف أو الفولدر (زي اللي يقدر يعدل أو يشغل أو يقرأ).

			- sudo
			- sudo apt update
			- sudo apt install ....
			- sudo apt upgrade -y
			- sudo apt dist-upgrade -y
			- sudo apt full-upgrade -y
